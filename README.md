## Installation

Install with npm

```bash
    git clone https://github.com/DutaKey/whatsapp-handler-template
    cd whatsapp-handler-template
    npm install
```

Rename `.env.example` to `.env` or create new file called `.env`.<br />
And, Input Your `Mongo_URI` and `DB_NAME`.<br />
Save, And Start Code.


Start Use
```
npm run start
# or
node ./index.js
```
